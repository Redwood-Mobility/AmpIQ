# AmpIQ Website

Single-page website for [AmpIQ](https://ampiq.tech), a hardware-agnostic EV charging consulting firm.

## Hosting

Hosted on GitHub Pages with custom domain `ampiq.tech`.

## Structure

```
├── index.html      # Single-page site
├── CNAME           # Custom domain config
├── assets/
│   ├── ampiq-logo.png    # Full wordmark logo
│   └── ampiq-icon.png    # Icon/mark only
└── README.md
```

## Local Development

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```
